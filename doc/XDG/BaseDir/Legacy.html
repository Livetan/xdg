Overview

Legacy API can serve as a stop gap until a developer has time to update an program already using XDG.

Do NOT use this module for future development!!!

def cache
  obj = XDG['CACHE']
  class << obj
    def home
      XDG['CACHE_HOME'].to_a.first
    end
    def dirs
      XDG['CACHE_DIRS'].to_a
    end
    def work
      XDG['CACHE_WORK'].to_a
    end
  end
  return obj
end
